# **App Name**: SMK LPPMRI 2 Kedungreja Website

## Core Features:

- Homepage Hero Section: Display a captivating headline, welcome message, and a strong call-to-action with a dynamic background.
- Vocational Programs Showcase: Highlight key majors with icons, brief descriptions, and links to detailed program pages.
- Admissions Information: Provide clear information on the admission process, requirements, and online registration form.
- Admin Dashboard Overview: Display key metrics, recent activity, and quick actions for school management.
- Content Management System: Enable full CRUD operations for website content (pages, news, events, gallery).
- AI-Powered Chatbot: Implement an intelligent chatbot to answer common questions from website visitors. The chatbot will use tool use to decide how to best incorporate various documents in the knowledge base when assembling its answer.

## Style Guidelines:

- Primary color: Deep navy blue (#1A237E) for professionalism and trust.
- Background color: Light gray (#F5F5F5), a slightly desaturated version of the primary, for a clean and modern look.
- Accent color: Vibrant teal (#00BFA5) for calls to action and highlights to inject energy.
- Body font: 'Inter', sans-serif, for modern and readable body text.
- Headline font: 'Poppins', sans-serif, to pair with 'Inter', for a contemporary, precise feel.
- Use a consistent set of modern, clean icons from Heroicons or Font Awesome.
- Incorporate smooth scroll animations, hover effects, and micro-interactions for enhanced user engagement.
